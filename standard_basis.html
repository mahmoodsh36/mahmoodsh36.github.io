standard basis

2024-02-16

the standard basis of a vector space is the set of vectors whose components are all zero, except one that equals 1.

consider the set which is a standard basis of the vector space also the dimension of this set is 2

when put in a matrix, a standard basis forms an identity matrix

consider the set which is a standard basis of the vector space we put the vectors in a matrix: which is an identity matrix of the 3rd degree
