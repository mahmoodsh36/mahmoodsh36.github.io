gradient descent

2023-04-22

starting at an arbitrary point in a function, we would like to make a step downwards towards a local minima, to do so we would need to find the slope of the function, which we do by finding its derivative, if the derivative at the point we're evaluates to a positive value, then the function must be ascending and so we need to go backward to go down, if it evaluates to a negative value, the function must be descending, and we need to move forward to go down, we dont need to worry about having to go upward/downward, as finding the derivative and placing it in an equation for later substitution is enough, the derivative would have the corresponding negative/positive sign anyway when we substitute the point we're at, so we just subtract that value (slope) from the variable on the axis, we often multiply the slope (before subtraction) we find by a constant referred to as the learning rate (usually a fraction) to make steps smaller so we dont overshoot the local minima point.
