discrete function

2023-06-13

a discrete function is one whose input and output (domain and image) is limited to a countable set of distinct, separate values. in other words, for every input value from the function's domain, there is a specific, discrete output value in the range. discrete functions are defined at isolated points in their domain, and there are gaps or discontinuities between these points.

discrete doesnt mean finite, although finite implies discrete
